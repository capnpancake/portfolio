PetSupplies.com

An e-commerce site created by myself and a team of classmates. The product of our capstone course for Chattahoochee Tech's Associate's of Applied Science in Computer Programming program. This application has a customer side for creating customer accounts, filling shopping carts, checking out - as a guest or returning customer, etc. There is also an application for the server side in which a hypothetical employee can login and see fulfilled and unfulfilled orders and prepare them for shipment or return.

  • Heavy focus on Java servlets and JSPs
  • HTML5 and CSS3
  • Microsoft Access Database and SQL
